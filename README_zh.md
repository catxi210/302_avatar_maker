- Next.js 14

## 开发&部署
1. 克隆项目 `git clone https://github.com/302ai/302_avatar_maker`
2. 安装依赖 `pnpm install`
3. 配置302的API KEY 参考.env.example
4. 运行项目 `pnpm dev`
5. 打包部署 `docker build -t coder-generator . && docker run -p 3000:3000 coder-generator`
